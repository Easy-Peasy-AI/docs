## Quick Example

```bash
curl -X POST https://easy-peasy.ai/api/generate \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "preset": "blog-post",
    "keywords": "How to get started with AI",
    "tone": "professional",
    "outputs": 1
  }'
```
